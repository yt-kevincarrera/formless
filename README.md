## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```


## Usage

1. **Build Your Form** - Use the component library to add form fields
2. **Configure Properties** - Customize labels, validation, and options
3. **Preview** - See your form in action
4. **Export** - Download or copy the generated code

## Export Options

### Quick Export

- Copy individual files (Component, Schema, Setup)
- Export form schema as JSON
- Download complete project bundle as ZIP

### CLI Install

- One-command installation for Bash/PowerShell
- Automatically creates files in your project
- No manual copy-paste needed

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - Component library
- **React Hook Form** - Form management
- **Zod** - Schema validation
- **Zustand** - State management
- **React Grid Layout** - Drag and drop
